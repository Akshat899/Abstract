Abstract — Brand Agency Landing Page

A modern brutalist-style brand agency landing page built using HTML, CSS, and JavaScript.
This project demonstrates advanced UI interactions, smooth scroll-triggered animations, and a dynamic portfolio slider designed for creative agencies.

The design focuses on bold typography, strong contrast, and motion-driven storytelling.

Live Preview
https://akshat899.github.io/Abstract/



Features
Hero Logo Animation

A floating logo animates from the navigation into the center of the hero section and returns to the header as the user scrolls.

Scroll Reveal Animations

Sections reveal smoothly as they enter the viewport using the Intersection Observer API.

Interactive Work Slider

A dynamic portfolio slider including:

Arrow navigation

Pagination dots

Swipe gestures on mobile

Background color transitions

Subtle parallax hover effects

Animated Statistics

Counters animate when the stats section becomes visible.

Performance Graph Bars

Animated bar graphs dynamically expand when scrolled into view.

Split Contact Section

A modern two-column animated contact layout with a sliding panel effect.

Mobile Navigation

Responsive hamburger navigation with smooth open/close transitions.

Fully Responsive

Optimized layouts for:

Desktop

Tablet

Mobile devices
